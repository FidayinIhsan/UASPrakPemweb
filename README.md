# Catatan Inventaris Organisasi (Laravel)

Skeleton project for CRUD inventory app built with Laravel 11 and Tailwind CSS.

## Cara Pakai
1. **Buat project Laravel baru**
   ```bash
   composer create-project laravel/laravel inventaris
   ```
2. **Salin file skeleton**
   Ekstrak seluruh isi folder ini ke dalam project Laravel Anda, menimpa jika diperlukan.
3. **Install Tailwind CSS** (opsional)
   ```bash
   npm install
   npm run dev
   ```
4. **Migrasi database**
   ```bash
   php artisan migrate
   ```
5. **Jalankan aplikasi**
   ```bash
   php artisan serve
   ```

---
Struktur folder ini memuat:
- Model, Controller, Migration
- Blade views dengan Tailwind
- Route resource lengkap
